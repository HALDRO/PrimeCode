## [Общий доступ](https://opencode.ai/docs/ru#%D0%BE%D0%B1%D1%89%D0%B8%D0%B9-%D0%B4%D0%BE%D1%81%D1%82%D1%83%D0%BF)
Разговоры, которые вы ведете с opencode, можно [поделиться с вашим команда](https://opencode.ai/docs/share).
```

/share

```

Это создаст ссылку на текущий разговор и скопирует ее в буфер обмена.
По умолчанию общий доступ к беседам не предоставляется.
Вот [пример диалога](https://opencode.ai/s/4XP1fce5) с opencode.
