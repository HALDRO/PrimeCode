## [Использование темы](https://opencode.ai/docs/ru/themes#%D0%B8%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5-%D1%82%D0%B5%D0%BC%D1%8B)
Вы можете выбрать тему, вызвав выбор темы с помощью команды `/theme`. Или вы можете указать это в файле [config](https://opencode.ai/docs/config).
opencode.json```

{



"$schema": "https://opencode.ai/config.json",




"theme": "tokyonight"



}

```
