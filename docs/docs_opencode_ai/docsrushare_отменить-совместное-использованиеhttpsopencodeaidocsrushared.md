## [Отменить совместное использование](https://opencode.ai/docs/ru/share#%D0%BE%D1%82%D0%BC%D0%B5%D0%BD%D0%B8%D1%82%D1%8C-%D1%81%D0%BE%D0%B2%D0%BC%D0%B5%D1%81%D1%82%D0%BD%D0%BE%D0%B5-%D0%B8%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5)
Чтобы прекратить делиться беседой и удалить ее из общего доступа:
```

/unshare

```

Это приведет к удалению ссылки общего доступа и удалению данных, связанных с разговором.
