## [Конфигурационный файл](https://opencode.ai/docs/ru/web#%D0%BA%D0%BE%D0%BD%D1%84%D0%B8%D0%B3%D1%83%D1%80%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D1%8B%D0%B9-%D1%84%D0%B0%D0%B9%D0%BB)
Вы также можете настроить параметры сервера в файле конфигурации `opencode.json`:
```

{



"server": {




"port": 4096,




"hostname": "0.0.0.0",




"mdns": true,




"cors": ["https://example.com"]



}


}

```

CLI-флаги имеют приоритет над настройками файла конфигурации.
