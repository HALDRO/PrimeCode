## [Настроить](https://opencode.ai/docs/ru#%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B8%D1%82%D1%8C)
С opencode вы можете использовать любого поставщика LLM, настроив его ключи API.
Если вы новичок в использовании поставщиков LLM, мы рекомендуем использовать [OpenCode Zen](https://opencode.ai/docs/zen). Это тщательно подобранный список моделей, протестированных и проверенных opencode. команда.
  1. Запустите команду `/connect` в TUI, выберите opencode и перейдите по адресу [opencode.ai/auth](https://opencode.ai/auth).
```

/connect

```

  2. Войдите в систему, добавьте свои платежные данные и скопируйте ключ API.
  3. Вставьте свой ключ API.
```

┌ API key


│


│


└ enter

```



Альтернативно вы можете выбрать одного из других поставщиков. [Подробнее](https://opencode.ai/docs/providers#directory).
