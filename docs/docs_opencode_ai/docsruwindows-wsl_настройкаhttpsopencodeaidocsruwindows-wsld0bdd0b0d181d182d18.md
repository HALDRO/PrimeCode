## [Настройка](https://opencode.ai/docs/ru/windows-wsl#%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B0)
  1. **Установите WSL**
Если вы еще не сделали этого, установите WSL по [официальному руководству Microsoft](https://learn.microsoft.com/en-us/windows/wsl/install).
  2. **Установите opencode в WSL**
После настройки WSL откройте терминал WSL и установите opencode одним из [способов установки](https://opencode.ai/docs/).
Окно терминала```


curl-fsSLhttps://opencode.ai/install|bash


```

  3. **Запускайте opencode из WSL**
Перейдите в каталог проекта (к файлам Windows можно обращаться через `/mnt/c/`, `/mnt/d/` и т.д.) и запустите opencode.
Окно терминала```


cd/mnt/c/Users/YourName/project



opencode

```
