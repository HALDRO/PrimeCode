## [Формат](https://opencode.ai/docs/config#%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%82)
opencode поддерживает форматы **JSON** и **JSONC** (JSON с комментариями).
opencode.jsonc```

{



"$schema": "https://opencode.ai/config.json",



// Theme configuration



"theme": "opencode",




"model": "anthropic/claude-sonnet-4-5",




"autoupdate": true,



}

```
