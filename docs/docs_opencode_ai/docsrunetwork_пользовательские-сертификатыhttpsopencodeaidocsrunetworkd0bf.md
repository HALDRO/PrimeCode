## [Пользовательские сертификаты](https://opencode.ai/docs/ru/network#%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D1%8C%D1%81%D0%BA%D0%B8%D0%B5-%D1%81%D0%B5%D1%80%D1%82%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%82%D1%8B)
Если ваше предприятие использует собственные центры сертификации для HTTPS-соединений, настройте opencode, чтобы доверять им.
Окно терминала```


export NODE_EXTRA_CA_CERTS=/path/to/ca-cert.pem


```

Это работает как для прокси-соединений, так и для прямого доступа к API.
